#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
https://github.com/bensuperpc/docker-lacc.git
```
```sh
cd docker-lacc
```
#### Usage

```sh
./compiler.sh cproc (where project to build)
```
#### Build
```sh
make linux/amd64
```
